توضيح كلّ بند

المنصّة + لوحة التحكّم
تطوير الواجهة الأماميّة العامّة (Vue/Nuxt) ولوحة تحكّم إداريّة (Vue + Vuetify) متكاملة مع API خلفي (.NET 8) بنماذج صلاحيّات متعدّدة، إدارة محتوى، أخبار، فعاليّات، مباريات، لاعبين، إحصائيّات.
تطبيقات iOS و Android
تطبيقان أصليّان (Swift / Kotlin أو Flutter حسب التوصية)، نشر على المتجرين، إشعارات Push، مزامنة لحظيّة مع المنصّة، ودعم العضويّات والمحفظة والمتجر داخل التطبيق.
المتجر + الدفع + التبرّعات
كاتالوج منتجات، عربة، تسجيل خروج، تكامل مع Plutu، تتبّع طلبات، نظام تبرّعات بقنوات متعدّدة، وتقارير ماليّة شفّافة للإدارة.
العضويّات وتسجيل المشجّعين
نظام عضويّات مُتدرّج (عاديّة، ذهبيّة، بلاتينيّة)، بطاقات مشجّع رقميّة، تجديد آلي، تحقّق هويّة بالرقم الوطني، ومحفظة نقاط ولاء.
الأرشيف وذاكرة الاتحاد
نظام أرشفة رقمي بأكثر من نمط (صور، فيديو، صحف، وثائق)، فهرسة زمنيّة، بحث متقدّم، عرض زمني تفاعلي، وحفظ طويل المدى مع تكرار.
التكاملات الخارجيّة
تكامل مع Odoo ERP (محاسبة، مخازن)، iSend للرسائل النصّية، بوّابة Plutu، APIs خارجيّة (KoraStats، مواقع التواصل)، وSSO حسب الحاجة.
الحماية والـ DevOps
Cloudflare (CDN + WAF + DDoS Protection)، شهادات TLS، نسخ احتياطي ثلاثيّ المواقع، خطّة استرداد كوارث، مراقبة 24/7، وأنابيب CI/CD.
التدريب والتوثيق
جلسات تدريب لفِرَق النادي (إدارة، إعلام، تسويق، خدمة الجمهور)، دليل مستخدم عربي، دليل فنّي للمشرفين، فيديوهات تعليميّة، ودعم انتقالي.
